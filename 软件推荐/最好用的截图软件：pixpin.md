## 功能强大，操作简单

Pixpin 拥有丰富的截图功能：区域截图、滚动截图，快捷键微调区域，截图固定到屏幕，还能制作成 git 或 video。

## 内置编辑工具，省时高效

截图后，直接对截图进行标注序号、裁剪、添加文字或涂鸦。

无论你使用的是 Windows 还是 macOS，Pixpin 都能完美兼容，并通过云端实现同步。
